## このゲームについて

ピンボールとブロック崩しの合いの子です。  
![ゲーム中の画面](playing_scene.jpg)

## ゲームのルール

1. トリガーでボールをはじく
1. ボールに当たったブロックは消える
1. ボールを落とさずにブロックを全て消したらGAME CLEAR！
1. ボールを落としてもリトライできます

## プレイ中の操作方法

![トリガー説明画像](trigger_detail.jpg)

- キーボード【←】：左トリガーが回転  
- キーボード【→】：右トリガーが回転
